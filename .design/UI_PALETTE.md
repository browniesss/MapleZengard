# UI 리소스 팔레트 — 메이플/단풍의전설 스타일

> MSW 협업 월드(모아요) 5차 기획서 톤(메이플 카툰풍 / 나무·버섯 / 헤네시스 녹색)에 맞춰
> `msw-search` REST 파이프라인(`msw_resource_api.cjs`)으로 수집한 실제 RUID 팔레트.
> 모든 RUID는 검색 결과 원본이며 날조 없음. `sprite` 타입 → UI `SpriteGUIRendererComponent.ImageRUID`에 직접 할당 가능.
> 9-slice가 필요한 패널/버튼/프레임은 `ImageType = Sliced` 권장.
> 좌향 정렬 컨벤션은 UI 정적 스프라이트엔 무관(월드 엔티티에만 해당).

## 비고 / 검증 메모

- **MISSING 항목 2건**: (1) "레드메소" 전용 빨간 메소 아이콘은 인덱스에 없음 → 색조 근접한 "브론즈 메소주머니"(`94b7d8…`)로 대체. (2) "자물쇠" 전용 padlock 아이콘 검색 실패(`lock/padlock/잠금` 모두 패널 스트립만 반환) → etc 아이콘(`130d27…`)으로 임시 대체. 실제 적용 전 `getResource`로 썸네일 확인 권장.
- **9-slice 권장**: 패널(`46de62…`, `d86e65…`), 일반/큰 버튼(`2a6302…`, `6dc2b5…`), 카드프레임(`33e8bc…`)은 가변 크기 → `ImageType = Sliced` + 적절한 border 설정.
- **직업 상징은 skill 카테고리 스킬 아이콘**으로 대체(전용 "직업 엠블럼" 리소스는 인덱스에 없음). 32x32 정사각이라 슬롯/탭 아이콘으로 적합.
- **HP/에너지 바**는 단일 스프라이트라 채움 연출은 `Slider` 또는 `FillAmount`(Filled ImageType)로 별도 fill 스프라이트를 겹쳐 구성 필요. 위 RUID는 프레임/배경 용도.
- 모든 RUID는 `searchResources`(POST /v3/search/resources) 결과 원본. 적용 전 각 RUID `get <ruid>`로 `payload.thumbnail` 시각 확인 권장.
- 검색 경로: `.claude/skills/msw-search/scripts/msw_resource_api.cjs` (REST 파이프라인). `asset_search_resources` 직접 호출 미사용.
